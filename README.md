# AI: The Past and The Present - ELIZA vs LLM Comparison

## 📚 Project Overview
This project implements and compares two different AI systems:
- **ELIZA (1966)** - A rule-based psychotherapy chatbot using pattern matching
- **Modern LLM** - A generative language model using transformer architecture

## 🎯 Learning Objectives
- Understand rule-based vs generative AI approaches
- Compare response patterns between past and present AI
- Analyze strengths and limitations of each system
- Gain practical experience with AI implementation

## 🚀 Getting Started

### Prerequisites
- Python 3.7 or higher
- pip package manager
